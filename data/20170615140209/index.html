Registered 2011-10-23 by Märt Põder

Simple context menu file comparison extension for Nautilus 3 and above, inspired by the discontinued 'diff-ext' extension. By default it uses 'meld' to do the comparison and provides "Compare", "Compare to ~/foo/bar" and "Compare Later" in Nautilus context menu. Using the setup utility you can choose your favourite compare tool for one-on-one, three-way and multi-compare situations.

You can find the PPA at https://launchpad.net/~boamaod/+archive/nautilus-compare. Usage of the context menu is described at http://diff-ext.sourceforge.net/gdiff-ext-guide.shtml.

About the status of diff-ext in Nautilus 3, please see http://bugs.debian.org/cgi-bin/bugreport.cgi?bug=637314
