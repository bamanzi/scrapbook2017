      About this Add-on

      For easy to switch to the other group's tab.

      == Pano Sidebar ==
      Show tabs tree by each groups on sidebar.
      • can switch to the tab by double-click on a tab item
      • can edit the group name by double click on a group item
      • can drag & drop tabs items or a group
      • can search title and URL in tabs
        • * is wild card
        • | is 'or' condition

      == Pano button ==
      toggle open/close Pano sidebar. And switch to another group from this menu.

      == All-tabs-button ==
      Add menus for switching the other group's tab on right hand side of the tab-bar.

      == Pano panel ==
      it's almost same as Pano sidebar, but a popup panel. You can use this temporarily.

      == Toolbar buttons ==
      Pano sidebar and panel have customizable toolbar.
      By default, the sidebar has a filter box and a collapse tabbar button and the panel has a filter. You can customize the toolbar from the context menu. Following items are available:
      • filter: show only matches the title or the URL
      • back/forward tab selection history button
      • all collapse/expand groups button
      • collapse the tabbar when the sidebar is appeared
      • close all blank groups

      == CSS Editor==
      Can edit CSS from "CSS Editor" in Pano option and change looks.
