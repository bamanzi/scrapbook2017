      About this Add-on

      This extension adds a submenu to the selection context menu. Opening the submenu shows a number of buttons of which the last opens DuckDuckGo and fills the search field with the selected text, a space and an exclamation mark. The other buttons allow you to search the selection using preconfigured !bangs, editable using preferences. More information about the !bang syntax can be found at https://duckduckgo.com/bang.html.

      Version Information

      Version 0.3 Released March 17, 2014 10.2 kB Works with Firefox 24.0 and later

      New: Changes in preferences process immediately (as a result, adding/removing !bangs does not require a restart).
